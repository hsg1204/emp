# Phase 1 Home and My Tasks

## Goal
Implement the phase-1 home dashboard and My Tasks / History Tasks pages on top of the existing frontend scaffold.

## Requirements
- Build the home page into a more realistic phase-1 dashboard using Ant Design and Tailwind.
- Keep the page focused on phase-1 product skeleton needs: summary cards, notice area, and lightweight overview sections.
- Implement My Tasks and History Tasks pages as real list pages rather than placeholders.
- Reuse common page patterns already established in the frontend scaffold.
- Keep interactions frontend-only with mock or static data.

## Acceptance Criteria
- [ ] Home page is no longer a basic placeholder and presents a usable dashboard layout.
- [ ] My Tasks page is implemented as a usable list page.
- [ ] History Tasks page is implemented as a usable list page.
- [ ] Loading / empty / error states are handled where relevant.
- [ ] `npm run build` passes in `frontend/`.

## Technical Notes
- Development type: frontend
- Follow the established layout, router, React Query, Ant Design, Tailwind, and shared list-page patterns already present in `frontend/src`.
- Do not introduce backend integration yet; keep this task mock-data driven.
